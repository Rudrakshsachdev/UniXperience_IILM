**UniXperience** is a premium, all-in-one digital gateway designed specifically for the students, faculty, and staff of IILM University. It transforms the complex campus ecosystem into a seamless, intuitive, and visually stunning digital experience.

---

## 🌟 The User Experience

At its core, UniXperience is built to simplify campus life. No more hunting through multiple portals or missed emails—everything you need is right here.

### 🚀 Key Features

#### 1. Dynamic Hero Portal
Your journey starts with a breathtaking visual experience. The dynamic hero section provides a localized, professional greeting that sets the tone for a modern academic environment.

#### 2. Intelligent Quick Access
Time is your most valuable resource. Our **Quick Access** dashboard brings your most essential links to the forefront:
- **📅 Class Schedule**: Your real-time daily timetable at your fingertips.
- **📚 Course Syllabus**: Instant access to learning objectives and materials.
- **📇 University Directory**: Connect with faculty and peers in seconds.

#### 3. The IILM Advantage
Discover why IILM stands out. Our **Why Choose Us** section highlights the pillars of our institution:
- **Academic Excellence**: Centralized resources for your success.
- **Event Management**: From hackathons to seminars, stay in the loop.
- **Campus Community**: A thriving, connected ecosystem of learners.

#### 4. Real-time News & Updates
Stay informed with a magazine-style news feed. Whether it's a guest lecture or a campus holiday, you'll see it first on UniXperience.

## 🛠️ Built for Performance

We believe that a university portal should be as fast as it is beautiful.

## 💻 Developer Quick Start

For those looking to run the project locally or contribute:

### Prerequisites
- Node.js (v18+)
- Python 3.10+

### Step-by-Step Setup

1. **Frontend**:
   ```bash
   cd frontend/uni-expirence
   npm install
   npm run dev
   ```
   Open `http://localhost:3000` to view the portal.

2. **Backend**:
   ```bash
   cd backend
   # Set up your virtual environment
   python -m venv myenv
   source myenv/bin/activate # or myenv\Scripts\activate on Windows
   pip install -r requirements.txt # (Ensure you have a requirements.txt)
   python main.py
   ```
